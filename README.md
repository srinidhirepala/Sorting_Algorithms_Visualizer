## Features

- **Clean Web Interface**: Simple and intuitive design
- **Sorting Algorithms**: Merge Sort and Quick Sort
- **Real-time Animation**: Watch algorithms step by step
- **Interactive Controls**: Adjust speed and array size
- **Statistics**: Track comparisons, swaps, and time

## Quick Start

1. **[Open Live Demo](https://yourusername.github.io/Sorting_Algorithm_Visualiser/)**
2. Or download and open `index.html` in any web browser
3. Click "Generate Array" and select a sorting algorithm

## How to Use

1. **Generate Array**: Click to create a random array
2. **Choose Algorithm**: Select Merge Sort or Quick Sort
3. **Adjust Settings**: Use sliders to control speed and size
4. **Watch**: Observe the sorting process with color-coded bars

## Color Coding

- **Blue**: Normal elements
- **Red**: Elements being compared
- **Green**: Sorted elements
- **Yellow**: Pivot element (Quick Sort)
- **Purple**: Elements being merged (Merge Sort)

## Algorithms

### Merge Sort
- **Time Complexity**: O(n log n)
- **Space Complexity**: O(n)
- **Stable**: Yes

### Quick Sort
- **Time Complexity**: O(n log n) average, O(n¬≤) worst
- **Space Complexity**: O(log n)
- **Stable**: No
